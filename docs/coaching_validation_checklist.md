## 2. Exercise Selection Quality

### Compound-to-Isolation Ratio
- [ ] Each session starts with 1–2 heavy compound movements
- [ ] Isolation work follows compounds (never the reverse for the same muscle)
- [ ] Ratio is approximately 40% compound / 60% isolation for hypertrophy blocks

### Movement Pattern Coverage (per muscle)
- [ ] **Chest**: flat press + incline press + fly/cable variation (stretch + squeeze)
- [ ] **Back**: vertical pull (pulldown/pull-up) + horizontal row + isolation (pullover/straight-arm)
- [ ] **Shoulders**: overhead press + lateral raise + rear delt fly (all 3 heads every week)
- [ ] **Quads**: knee-dominant compound (squat/leg press) + extension isolation
- [ ] **Hamstrings**: hip hinge (RDL/SLDL) + knee flexion (leg curl)
- [ ] **Biceps**: supinated curl + neutral/hammer curl + stretched position curl (incline)
- [ ] **Triceps**: overhead extension (long head) + pushdown (lateral head) + close-grip press

### Exercise Variety
- [ ] No more than 2 exercises repeated between sessions for the same muscle
- [ ] Exercises vary across mesocycles (no stale movement patterns)
- [ ] No Olympic lifts (clean, snatch, jerk, push press)

| Load Zone | Rep Range | % of Total Volume | Check |

| Strength-Hypertrophy | 5–8 | ~20% | [ ] |
| Core Hypertrophy | 8–12 | ~50% | [ ] |

- [ ] Heavy compounds (squat, bench, row, RDL) are in the 5–8 rep range
- [ ] Isolation movements are in the 10–15+ rep range
- [ ] FST-7 finishers use 8–15 reps (7 sets, short rest)

---

## 4. Equipment & Load Type Distribution

- [ ] ≤40% of session exercises are free-weight axial (barbell squat, deadlift, OHP)
- [ ] Machines and cables make up ≥50% of total exercise count for Men's Open
- [ ] Dumbbells are used for unilateral work and stretch-position movements
- [ ] Exercise database tag for load type (plates, cable, machine, dumbbell) matches real gym equipment

---

## 5. Split Design & Frequency

### Weekly Frequency Per Muscle
- [ ] All muscles trained at least 1x/week
- [ ] Priority (lagging) muscles trained 2x/week
- [ ] Muscles with HQI gap >15 cm forced to 2x/week minimum
- [ ] No muscle trained >3x/week (recovery constraint)

### Session Structure
- [ ] Training days per week matches user preference (4–6d typical for pro)
- [ ] Antagonist or synergist pairing is logical (no chest + back + legs in one session)
- [ ] Session volume is 16–24 working sets per session (absolute max 28)
- [ ] Session duration estimate is 60–90 minutes (not 2+ hours)
- [ ] Split is truly customised, not defaulting to generic PPL

---

## 6. Progressive Overload Model

### Starting Weights
- [ ] All exercises have a non-null starting weight (seed weight or from baselines)
- [ ] Seed weights are conservative (~RPE 6–7) to prevent overloading new users
- [ ] Weight increments match equipment type:
  - Barbell: 2.5–5 kg steps
  - Dumbbell: 2.5 kg steps
  - Cable/Machine: 2.5 kg (pin stack)
  - Plate-loaded machine: 5–10 kg steps

### Progression Logic
- [ ] Double progression: reps increase first, then weight resets reps to floor
- [ ] RPE gate: weight only increases when RPE ≤ 8 at rep ceiling
- [ ] Consolidation: RPE ≥ 9.5 holds weight and reps (no forced increases)
- [ ] Progressive overload applies session-to-session, not just week-to-week

---

## 7. Warm-Up Protocol

- [ ] First exercise per muscle group gets warm-up sets
- [ ] Warm-up scheme follows ramping percentages (40% → 60% → 75% → 85%)
- [ ] Warm-up reps decrease as load increases (10 → 5 → 3 → 1)
- [ ] Subsequent exercises for same muscle skip warm-ups (already primed)

---

## 8. Fatigue Management & Recovery

### Within Session
- [ ] CNS-heavy compounds are placed first in each session
- [ ] No consecutive heavy axial loading (e.g. heavy squats → heavy deadlifts)
- [ ] SFR (Stimulus-to-Fatigue Ratio) is weighted: exercises with SFR < 0.5 are limited to isolation slots

### Across the Week
- [ ] 48-hour minimum gap between sessions for same muscle group
- [ ] Back and hamstrings share days carefully (shared spinal erector load)
- [ ] Shoulder pressing and chest pressing are spaced by ≥1 day

### Mesocycle Level
- [ ] Volume ramps: MEV (week 1) → MAV (weeks 2–4) → MRV (week 5) → Deload (week 6)
- [ ] Deload week reduces volume to ~50% and intensity to RPE 6
- [ ] Soreness check-in modifies daily volume caps (dynamic, not static)

---

## 9. FST-7 Implementation

- [ ] Exactly 7 sets where an FST-7 finisher is flagged
- [ ] FST-7 is only applied to isolation/machine movements (never compound barbell)
- [ ] Rest periods for FST-7 sets are 30–45 seconds (not compound rest times)
- [ ] Maximum 1–2 FST-7 finishers per session
- [ ] FST-7 targets the priority muscle of the session

---

## 10. Bodybuilding-Specific Coaching Benchmarks

### Common Coaching Red Flags
- [ ] No exercise is prescribing >6 working sets unless it's FST-7
- [ ] No session has >28 total working sets
- [ ] No session exceeds 2 heavy barbell compounds
- [ ] Lateral raises appear at least 2x/week for shoulder width
- [ ] Upper chest (incline) appears at least 1x/week
- [ ] Rear delts appear at least 1x/week
- [ ] Hamstrings are not just an afterthought on leg day

### Athlete-Specific Adjustments
- [ ] Division-specific exercise selection (Men's Open prioritises mass, Classic prioritises aesthetics)
- [ ] Injury contraindications are respected (no behind-the-neck press with shoulder impingement)
- [ ] Bilateral asymmetry >5% triggers unilateral preference
- [ ] Equipment availability constraints are honoured

4. Feed violations back as algorithmic patches to the engine
5. Re-generate and re-validate until all checks pass
